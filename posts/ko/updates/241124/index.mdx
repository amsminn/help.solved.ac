안녕하세요! 우리 모두가 만들어가는 알고리즘 문제해결 학습의 이정표 **solved.ac**입니다.

최근 변경된 X (구 Twitter) API 정책에 따라, solved.ac 자동 포스트 기능은 전체 유저를 통틀어 24시간당 17개 포스트만을 할 수 있도록 제한되었습니다.

이로 인하여 solved.ac는 **2025년 1월**부터 자동 포스트 기능을 제공하지 않을 예정이며, 추후 다른 마이크로블로그 서비스에서 유사 기능을 제공할 수 있도록 하겠습니다.
