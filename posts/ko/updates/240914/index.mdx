안녕하세요! 우리 모두가 만들어가는 알고리즘 문제해결 학습의 이정표 **solved.ac**입니다.

BOJ의 문제 출처 기능을 솔브드의 고급 검색과 더 쉽게 연동할 수 있게 하기 위한 **대회 문제 아카이브** 페이지를 소개합니다.

## 문제 아카이브

<ExternalLink href="https://solved.ac/problems/archive/">문제 아카이브</ExternalLink>

솔브드의 [고급 검색](https://solved.ac/search) 기능은 `/<contest-name>`을 통하여 출처 기반 필터링이 가능하나, `<contest-name>`에 들어갈 대회 ID를 찾기는 어려웠습니다.
이를 보완하기 위하여 대회 문제 아카이브 페이지를 마련하였습니다.

문제 아카이브 페이지에서는 문제 출처 ID를 쉽게 찾을 수 있고, 대회 ID 필터가 입력된 검색 페이지로 빠르게 이동할 수 있어, 대회 ID에 다른 필터들을 조합하기 쉬워집니다.
또한, 솔브드의 유저 인터페이스를 기반으로 하여, 문제 기여 및 북마크 등의 기능에 더 편리하게 액세스할 수 있습니다.
